## BD Stuff

Set of plugins and themes created for _Better Discord_.

### Plugins

- [BetterDateViewer](https://github.com/ezeholz/BDStuff/tree/main/Plugins/BetterDateViewer) displays current time, date, and day of the week on the bottom of the member list. The way it is formatted depends on your locale conventions. It supports both dark and light theme.
- [BetterUpdater](https://github.com/ezeholz/BDStuff/tree/main/Plugins/BetterUpdater) automatically checks for updates for non-official BetterDiscord plugins from GitHub repositories.
- [Date Viewer](https://github.com/ezeholz/BDStuff/tree/main/Plugins/DateViewer) deprecated as the updater now wrongly tries to update it.
